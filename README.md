## Objetivo 🎯

Crear una página web responsive de un restaurante utilizando Bootstrap con tres apartados principales: Inicio, Carta y Contacto.

## Sobre el proyecto 🔎

Es una página web de un restaurant sencilla hecha con Bootstrap incluyendo sus diferentes apartados como el Inicio, Carta y Contacto.
